Frequently asked

Common questions

How do I connect my LitterCheck sensor to the app?

To connect your sensor, open LitterCheck and navigate to the Monitor tab. If no device is set up, you'll be taken to the Connect screen automatically.

  1. Tap Connect a Device.
  2. Enter your sensor's Device ID (printed on the back of the hardware unit, or in your device's firmware configuration).
  3. Give your litter box a name and choose an icon.
  4. Tap Connect — LitterCheck will verify the device ID and begin streaming data.

If you don't have a hardware sensor, you can use Manual Mode instead — tap "Set Up Manually" on the Connect screen.

The app says "Could not load data" — what do I do?

This usually means the app can't reach the Firebase database. Try the following steps in order:

  1. Check your internet connection — the app needs network access to fetch sensor data.
  2. Tap Retry — use the Retry button shown on the error screen to re-attempt the connection.
  3. Check your sensor is powered on — if the hardware unit is off or not connected to Wi-Fi, it won't be publishing data.
  4. Pull to refresh — on the Monitor tab, pull down to force a refresh of the data stream.

If the issue persists after all of the above, the device ID may have been entered incorrectly. Go to Settings → Remove Device and re-add it with the correct ID.

What do the VOC and IAQ readings mean?

VOC (Volatile Organic Compounds) are gases released by decomposing waste — primarily ammonia. The sensor measures these continuously using a metal-oxide (MOX) element.

IAQ (Indoor Air Quality index) is a derived score calculated from VOC readings. It runs from roughly 0 (clean air) to 500+ (very poor air). A litter box at rest in a clean state typically reads below 150.

LitterCheck uses the trend and magnitude of VOC readings to determine whether the litter box is Clean (green) or Dirty (red). You can view the full history in Sensor Log, which is accessible from the Monitor tab.

How does the app handle multiple cats?

When you set up a device or manual litter box, you can specify how many cats use it. This affects the recommended cleaning interval:

  • 1 cat — 48-hour grace period after marking as cleaned
  • 2 cats — 24-hour grace period
  • 3+ cats — 12-hour grace period

After the grace period expires, the status reverts to whatever the sensor reports (or is shown as overdue in manual mode). You can update the cat count anytime in Settings → Device Settings.

Veterinarians recommend one litter box per cat, plus one extra. LitterCheck supports adding multiple devices — use the + Add Another Device option in the profile switcher.

Can I use LitterCheck without a sensor?

Yes — Manual Mode lets you use LitterCheck without any hardware sensor. It works as a smart cleaning tracker:

  • You log each cleaning session manually by tapping Mark as Cleaned.
  • The app tracks the time elapsed and shows when a clean is recommended based on your cat count.
  • Historical cleaning logs are stored on-device.

To set up manual mode, tap Connect a Device → Set Up Manually on the Connect screen. You can switch between manual and sensor-connected profiles from the device switcher at any time.

How do I add or switch between multiple litter boxes?

LitterCheck supports multiple devices and manual profiles in one app:

  1. On any tab, tap the device name at the top of the screen to open the device switcher.
  2. Tap Add Another Device at the bottom to connect a new sensor or create a new manual profile.
  3. Tap any device in the list to switch to it immediately.

Each device profile has its own name, icon, cat count, and history. You can rename or remove profiles in Settings.
